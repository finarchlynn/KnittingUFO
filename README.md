# 뜨개 프로젝트 기록 앱 (Knit Tracker)

나만의 뜨개 프로젝트를 기록하는 개인용 웹앱입니다. 도안·바늘·실·게이지 같은 정보와 사진을 한곳에 모으고, 프로젝트 시트를 PDF로 저장하거나 전체 목록을 CSV로 내보낼 수 있습니다.

## 주요 기능

- **프로젝트 기록**: 프로젝트명, 상태(찜한 도안·계획·진행중·완성), 유형, 계절, 도안 작가, 도안, 바늘, 실, 실 사용량, 게이지, 사이즈, 시작일·완성일, 메모
- **사진 첨부**: 프로젝트당 최대 12장 (자동 축소 저장)
- **필터**: 상태(전체 / 내 작업 / 도안 레이어) + 유형·계절·바늘·도안작가
- **PDF 저장**: 프로젝트별 시트를 사진 포함 PDF로 내려받기
- **CSV 내보내기**: 전체 프로젝트의 텍스트 정보를 CSV로 내보내 Google Sheets·Excel에서 열기

## 데이터 저장 방식

모든 데이터는 **사용하는 브라우저 안(IndexedDB)에만 저장**됩니다. 서버로 전송되지 않습니다.

- 같은 브라우저·기기에서는 새로고침해도 유지됩니다.
- 다른 기기·브라우저로는 자동으로 옮겨지지 않습니다. 기기 간 이동이 필요하면 CSV 내보내기를 백업으로 쓰세요. (단, 사진은 CSV에 포함되지 않습니다.)
- 브라우저 데이터를 삭제하면 함께 지워집니다.

## 로컬 실행

[Node.js](https://nodejs.org) 18 이상이 필요합니다.

```bash
npm install
npm run dev
```

터미널에 표시되는 주소(예: `http://localhost:5173`)를 브라우저에서 엽니다.

## 빌드 & 배포

```bash
npm run build      # dist/ 폴더에 정적 파일 생성
npm run preview    # 빌드 결과 미리보기
```

`dist/` 폴더를 정적 호스팅에 올리면 됩니다.

- **Vercel / Netlify**: 저장소를 연결하면 자동 빌드됩니다. (빌드 명령 `npm run build`, 출력 폴더 `dist`)
- **GitHub Pages**: `npm run build` 후 `dist/` 내용을 `gh-pages` 브랜치에 올리거나 GitHub Actions를 사용하세요. `vite.config.js`의 `base: "./"` 설정으로 하위 경로에서도 동작합니다.

## 휴대폰에서 앱처럼 쓰기

배포한 주소를 휴대폰 브라우저에서 연 뒤 **홈 화면에 추가**하면 앱처럼 전체 화면으로 실행됩니다.

## 메모

- PDF 생성은 실행 시점에 `jsPDF`·`html2canvas`를 CDN에서 불러옵니다(인터넷 연결 필요).
- 클라우드 동기화(Google Sheets 자동 연동)는 포함되어 있지 않습니다. 필요하면 Google Apps Script나 별도 백엔드를 붙이는 방향으로 확장할 수 있습니다.

## 기술 스택

Vite + React 18. 외부 UI 라이브러리 없이 단일 컴포넌트 파일(`src/App.jsx`)로 구성했습니다.
